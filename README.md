## 第二阶段

产后一周才能喝浓汤，比较适宜的汤是富含蛋白质、维生素、钙、磷、铁、锌等营养素的清汤，如瘦肉汤、蔬菜汤、蛋花汤、鲜鱼汤等，而且要保证汤和菜、肉一起吃，这样才能真正摄取到营养。

分娩中，新妈妈体力消耗大，胃肠肌张力及蠕动减弱，需要一周左右才能恢复，因此，不宜进食比较油腻的鸡汤、鱼汤等。一周后，再增加鸡汤、鱼汤等高蛋白、富有营养的汤汁食物，以帮助下奶。
